## 🎯 Ejercicio

**Archivo**: `functions.php`

**Tarea**: Cambia la regla de impuestos para que sea escalonada (ej. 10% hasta $100 y 21% por encima).

## Ejecutar tests

```bash
./vendor/bin/phpunit tests/CodeSmells/Dispensables/DuplicatedCodeTest.php
```

## Problema a experimentar

Tendrás que actualizar múltiples implementaciones y recordar mantenerlas consistentes, lo que demuestra cómo la duplicación multiplica el esfuerzo y el riesgo de error humano.
