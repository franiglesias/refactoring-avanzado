# Dispensables - Code Smells

Los **Dispensables** son code smells que representan código que no es necesario, que puede ser eliminado sin afectar la funcionalidad del sistema.

## Ejercicios incluidos

### 1. Comments
[Comments.java](Comments.java)

Comentarios que explican código obvio o que deberían ser reemplazados por código auto-explicativo.

### 2. Data Class
[DataClass.java](DataClass.java)

Una clase que solo contiene campos y getters/setters sin comportamiento propio.

### 3. Dead Code
[DeadCode.java](DeadCode.java)

Código que nunca se ejecuta o nunca se usa: variables, métodos o clases no utilizadas.

### 4. Duplicated Code
[DuplicatedCode.java](DuplicatedCode.java)

El mismo código aparece en múltiples lugares, violando el principio DRY.

### 5. Lazy Class
[LazyClass.java](LazyClass.java)

Una clase que no hace lo suficiente para justificar su existencia.
